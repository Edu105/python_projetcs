# 📊 Comparación de Duración de Cursos Python

Script en Python que compara la duración de un curso gratuito de Python en YouTube con otros cursos pagos, calculando el **porcentaje de tiempo ahorrado**.

## 🎯 Propósito ejercicio 1.1

Demostrar matemáticamente que el curso de YouTube (1.5h) es significativamente más eficiente que:
- Curso más corto de otros: **2.5h** (40% más corto)
- Curso promedio: **4h** (62.5% más corto)  
- Curso más largo: **7h** (78.6% más corto)

# 🗣️ Calculadora de Velocidad de Habla

Script interactivo que **estima el tiempo de habla** basado en 2 palabras/segundo promedio, compara con Johan (30% más rápido) y detecta textos largos.
